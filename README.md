#  AI Student Performance & Learning Intelligence System

##  Overview
This project is a Machine Learning system that analyzes student performance, predicts results, groups students, and simulates learning strategies.

It combines multiple AI concepts:
- Supervised Learning (KNN, SVM, Perceptron)
- Unsupervised Learning (K-Means Clustering)
- Dimensionality Reduction (PCA)
- Reinforcement Learning (Q-Learning simulation)

---

##  Objectives

✔ Predict student result (Pass/Fail)  
✔ Compare ML models performance  
✔ Group students using clustering  
✔ Visualize data using PCA  
✔ Simulate study strategy using Q-Learning  

---

##  Dataset

| Feature | Description |
|--------|------------|
| Hours_Study | Study time |
| Attendance | Class attendance |
| Assignments | Homework completed |
| Marks | Exam score |
| Result | Target (Pass/Fail) |

---

##  Machine Learning Models

### ✔ Supervised Learning
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Perceptron

### ✔ Unsupervised Learning
- K-Means Clustering

### ✔ Dimensionality Reduction
- PCA (Principal Component Analysis)

### ✔ Reinforcement Learning
- Q-Learning (Study strategy simulation)

---

##  Project Workflow

1. Load dataset
2. Train ML models
3. Evaluate accuracy
4. Perform clustering
5. Visualize using PCA
6. Simulate Q-learning
7. Predict new student result

##  Output Example
